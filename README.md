# M&M — Mist & Muse | Wear Confidence

A luxury fragrance store landing page and ordering platform based in Kumasi, Ghana. Featuring premium UAE-crafted fragrances, designer perfumes, a dynamic WhatsApp-based checkout flow, and an interactive review system.

---

## 🌟 Features

- **Curated Fragrance Collection**: Beautifully categorized layout for UAE Perfumes (GH₵200 & GH₵60 categories), Premium Body Mists (GH₵160), and Elite Designer Perfumes.
- **Seamless Checkout Flow**: Integrates directly with WhatsApp, generating personalized message templates dynamically populated with product details, pricing, and time-of-day appropriate greetings.
- **Interactive Customer Reviews**: Customers can filter reviews by rating, view verified badge statuses, like reviews, and submit new ratings. Custom feedback is stored locally via `localStorage`.
- **Ultra-Premium Aesthetics**: Sleek dark mode visual design built with custom HSL/OKLCH color palettes (Ink, Obsidian, Charcoal, Cream, Gold), subtle micro-animations (glow pulses, floating elements, shimmer effects), and glassmorphism.
- **SEO Optimized**: Fully responsive layout with structured title/meta tags, OpenGraph (OG) properties, semantic HTML, and high performance.

---

## 🛠️ Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/start/overview) (React 19, TypeScript, file-based routing)
- **Bundler & Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (using `@tailwindcss/vite` compiler plugin)
- **Animations**: `tw-animate-css` & custom CSS keyframes
- **Icons**: [Lucide React](https://lucide.dev/)
- **Component Primitives**: [Radix UI](https://www.radix-ui.com/)
- **Package Managers**: Bun & npm supported

## 🚀 Getting Started

### 📋 Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18+) or [Bun](https://bun.sh/) installed.

### 1. Install Dependencies

You can use either **Bun** or **npm** to install dependencies:

```bash
# Using Bun (recommended)
bun install

# Using npm
npm install
```

### 2. Run the Development Server

Start the local Vite development server with hot-module reloading:

```bash
# Using Bun
bun dev

# Using npm
npm run dev
```

The app will start at `http://localhost:3000` (or the next available port specified by Vite).

### 3. Build for Production

Build the production bundle:

```bash
# Using Bun
bun run build

# Using npm
npm run build
```

This compiles client and server assets, optimizing production bundles.

### 4. Preview the Production Build

```bash
# Using Bun
bun run preview

# Using npm
npm run preview
```

## 🧭 Routing Conventions

This project uses **file-based routing** handled by TanStack Router.

- All routing configurations are defined in the `src/routes/` directory.
- `src/routes/__root.tsx` acts as the root layout shell.
- `src/routes/index.tsx` renders the homepage.
- The route tree is automatically generated under `src/routeTree.gen.ts`. **Do not edit this file manually.**

## 🎨 Theme & Customization

The project uses custom luxury-brand CSS variables located in `src/styles.css`:

```css
--color-gold: #d4af37; /* Luxury gold accent */
--color-cream: #f5efe0; /* Warm white typography */
--color-charcoal: #1a1a1a; /* Secondary backgrounds */
--color-ink: #0a0a0a; /* Deep background */
```

Custom animations (e.g. `.animate-glow-pulse`, `.animate-shimmer`, `.animate-float`) are defined under `@layer base` and theme utilities for seamless usage throughout the codebase.
